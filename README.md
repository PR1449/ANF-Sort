# ANF-Sort
#### Сортировка по возрастанию реализована на безе MergeSort. Ключом является число которое является результатом сложения количества слагаемых со старшойстепенью многочлена Жегалкина.
#### Ключ special_num — это целое число, вычисляемое в функции ANF_special_number по следующему алгоритму:
1) Строится полином Жегалкина для булевой функции, заданной вектором f_vec (8‑битная строка).
- Используется метод треугольника Паскаля (build_pascal_triangle).
- Результат — строка target, где каждый символ '1' соответствует ненулевому коэффициенту в полиноме.
2) Определяется старшая степень полинома
- Для каждого слагаемого (где target[i] == '1') вычисляется количество переменных в конъюнкции.
- Например, для индекса i=5 (в двоичной записи 101) конъюнкция — x1x3 (2 переменные).
- Максимальная длина такой конъюнкции сохраняется в degree.
3) Считается количество слагаемых
4) Вычисляется special_num
#### Реализована структура Arr, в которой содержатся вводимые пользователям значения для сортировки (original_value) и значения специальных чисел.
#### MergeSort сортирует special_num. Выводятся отсортированные элементы original_value.

#### Асимптотика программы
- Ввод и вычисление ключей: O(n).
- Сортировка: O(n log n).
- Вывод: O(n).
- *Итоговая сложность: **O(n log n)**.*
####     Пространственная сложность:
- Массивы arr и buffer в MergeSort: O(n).
- Временные строки в функциях: O(1).
- *Итого: **O(n)**.*
